# rpa-jobs

Civilian job framework and employment loops.

## Features
- **Delivery Job**: Basic truck delivery loop (Depot -> GPS -> Deliver -> Paid).
- **Configurable**: Add new routes and pay rates in `config.lua`.

## Installation
1. Ensure `rpa-lib` is started.
2. Add `ensure rpa-jobs` to your `server.cfg`.

## Usage
- Go to the Job Center (Cityhall) or find the specific job NPC on the map to start.
